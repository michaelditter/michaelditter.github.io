Image Guide for Michael Ditter's Website

This guide helps you prepare and add the necessary images for your website. The website requires several types of images to be fully functional and visually appealing.

1. Profile/Headshot Image

Your profile/headshot image is used in the author bio section and about page.

  • Path: /img/profile/michael-ditter-headshot-new.jpg
  • Recommended size: 400px × 400px
  • Format: JPG or PNG

Tip: Use a clear, professional headshot with good lighting and a neutral background.

2. Blog Featured Images

Each blog post should have a featured image. Bitcoin market reports automatically generate their own, but other posts need custom images.

Bitcoin Report Images (Auto-generated)

  • Path: /img/blog/bitcoin-market-report-YYYY-MM-DD.jpg
  • Size: Generated automatically

Custom Blog Post Images

  • Path: /img/blog/[post-slug].jpg
  • Recommended size: 1200px × 630px
  • Format: JPG or PNG

Tip: Use high-quality, relevant images that represent the content of your post.

3. Favicon/Site Icons

These icons appear in browser tabs, bookmarks, and when the site is added to home screens.

  • Primary favicon: /img/icons/favicon.ico
  • Apple Touch Icon: /img/icons/apple-touch-icon.png (180px × 180px)
  • Android/General Icons:
    • /img/icons/icon-192.png (192px × 192px)
    • /img/icons/icon-512.png (512px × 512px)

Tip: You can use online favicon generators like realfavicongenerator.net to create a complete set of icons from a single image.

4. Social Media Icons

These are used in the footer and author bio sections.

The site is currently using Font Awesome icons, so you don't need to upload separate image files for these.

How to Add Images to Your Site

Option 1: Using GitHub Web Interface

  1. Go to your GitHub repository
  2. Navigate to the appropriate directory (e.g., img/profile/)
  3. Click "Add file" → "Upload files"
  4. Drag and drop your images or click to select them
  5. Add a commit message (e.g., "Add profile headshot")
  6. Click "Commit changes"

Option 2: Using Git Command Line

# Ensure you're in your repository directory
cd /Users/michaelditter/Michael\ Ditter\ Branding

# Add your image files to the appropriate directories
# For example, to add your headshot:
cp /path/to/your/headshot.jpg img/profile/michael-ditter-headshot.jpg

# Add other images similarly
cp /path/to/favicon.ico img/icons/favicon.ico
cp /path/to/touch-icon.png img/icons/apple-touch-icon.png
# etc.

# Stage the files
git add img/

# Commit the changes
git commit -m "Add website images"

# Push to GitHub
git push origin main
